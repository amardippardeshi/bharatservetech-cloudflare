# BHARAT SERVE TECH — WEBSITE RULES

## 1. TEMPLATE RULE
All pages must follow templates/master-template.html structure.

## 2. NO COPY PASTE RULE
Do not duplicate full HTML structure manually.

## 3. PAGE CONTRACT
Each page must contain:
- PAGE_TITLE
- PAGE_DESCRIPTION
- CANONICAL_URL
- PAGE_HEADING
- PAGE_CONTENT

## 4. SEO RULE
Every page must have:
- unique title
- unique description
- correct canonical URL

## 5. ENCODING RULE
All files must be UTF-8 only.

## 6. DRIFT RULE
If structure is modified in one page, update template first.

## 7. GOLDEN RULE
Make invalid state impossible before adding complexity.
